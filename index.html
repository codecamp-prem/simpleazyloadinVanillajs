Learn Java the Easy Way is a hands-on introduction to Java programming on Mac, PC, and Linux. Perfect for beginners, or for programmers of other languages who want to learn Java and Android. You will learn how to code desktop and mobile apps in Java right away; how to create fun, interactive apps like the animated, multi-touch BubbleDraw app, a Secret Messages app, and a Guessing Game; and you’ll master concepts like variables, loops, functions, and object-oriented programming concepts in a fun way with Java. “The teaching is clear, the graphics are engaging, and the apps are awesome. This is the programming course for parents and kids to enjoy together.” —Aaron Walker, Cybersecurity Expert, NASA
